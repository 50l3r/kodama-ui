# KEmpty
Con el componente `k-empty` puedes informar al usuario de que ahi no se le ha perdido nada, simplemente no hay nada que mostrar.

<Story name="Default" parameters={renderParameters({showPanel: true})} args={{title:'Sin resultados', subtitle: 'Prueba con otra consulta'}}>
  {Template.bind({})}
</Story>

<ArgsTable story="Default" />

## **Titulo / Subtitulo**
Puedes modificar los textos de titulo y subtitulo con las propiedades `title` y `subtitle` respectivamente.

<Canvas>
  <Story name="Title/Subtitle" argTypes={renderArgs(KEmpty, {}, true)} parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        components: { KEmpty },
        template: `
          <k-empty title="Titulo personalizado" subtitle="Subtitulo personalizado" />
        `,
      };
    }}
  </Story>
</Canvas>
