# KLogo
Con el componente `k-logo` puedes mostrar el logo de tu marca en distintas formas y colores.

<Story name="Default" parameters={renderParameters({showPanel: true})}>
  {Template.bind({})}
</Story>

## **Modo Oscuro**
Con el atributo `dark` o `light` puedes forzar el modo oscuro o claro del logo.

<Canvas>
  <Story name="Dark/Light" argTypes={renderArgs(KLogo, {}, true)} parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        components: { KLogo },
        template: `
          <div class="flex flex-row gap-6">
            <div class="bg-gray-800 p-5 rounded-lg">
              <k-logo dark />
            </div>
            <div class="bg-white p-5 rounded-lg">
              <k-logo light />
            </div>
          </div>
        `,
      };
    }}
  </Story>
</Canvas>

## **Side**
Con el atributo `side` se muestra el icono en formato rectangular en vez de cuadrado.
