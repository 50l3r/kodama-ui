# KTooltip
Con el componente `k-tooltip` puedes mostrar informaciÃ³n adicional sobre un elemento al pasar por encima o clicar sobre el.

<Story name="Default" parameters={renderParameters({showPanel: true})} args={{content:'Lorem ipsum dolor sit', placement: 'top'}}>

## **Titulo**
El titulo es el contenido del tooltip y puede especificarse con la propiedad `title` o el slot `title`.

<Canvas>
  <Story name="Title"  parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        template: `
          <k-tooltip content="Lorem ipsum dolor sit">
            <k-button class="mr-4">Propiedad</k-button>
          </k-tooltip>
          <k-tooltip>
            <k-button>Slot</k-button>
            <template #content>
              Lorem ipsum dolor sit
            </template>
          </k-tooltip>
        `,
      };
    }}
  </Story>
</Canvas>

## **Click/Hover**
Es posible modificar el compartamiento del tooltip para que se habra la propiedad `trigger`

<Canvas>
  <Story name="Method"  parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        setup() {
          const visible = ref(false);
          return {
            visible,
          };
        },
        template: `
          <k-tooltip trigger="mouseenter" content="Tooltip on hover">
            <k-button class="mr-2">HOVER</k-button>
          </k-tooltip>
          <k-tooltip trigger="click" hide-on-click content="Tooltip on click">
            <k-button class="mr-2">CLICK</k-button>
          </k-tooltip>
        `,
      };
    }}
  </Story>
</Canvas>

## **Flecha**
Puedes esconder la flecha del tooltip especificando la propiedad `arrow` a `false`

<Canvas>
  <Story name="Arrow"  parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        setup() {
          const visible = ref(false);
          return {
            visible,
          };
        },
        template: `
          <k-tooltip :arrow="false" content="Tooltip witouth arrow">
            <k-button>ARROW HIDE</k-button>
          </k-tooltip>
        `,
      };
    }}
  </Story>
</Canvas>
