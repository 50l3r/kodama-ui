# KCard
Con el componente `k-card` puedes crear tarjetas de contenido de forma sencilla sin especificar tu mismo las clases.

<Story name="Default" parameters={renderParameters({showPanel: true})}>
  {Template.bind({})}
</Story>

<ArgsTable story="Default" />

## **Header/Footer**
Puedes definir el `header` de la tarjeta con sus slot correspondiente.

<Canvas>
  <Story name="Header" argTypes={renderArgs(KCard, {}, true)} parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        components: { KCard },
        template: `
          <k-card>
            <template #header>
              Header Example
            </template>
            <k-skeleton type="card" />
          </k-card>
        `,
      };
    }}
  </Story>
</Canvas>

## **Footer**
Puedes definir el `footer` de la tarjeta con sus slot correspondiente.

<Canvas>
  <Story name="Footer" argTypes={renderArgs(KCard, {}, true)} parameters={renderParameters({showPanel: false})}>

      return {

        template: `
          <k-card>
            <k-skeleton type="card" />

            </template>
          </k-card>
        `,
      };
    }}
  </Story>
</Canvas>
