## **Icon**
Esta propiedad agrega un ícono al botón. El botón tendrá un aspecto cuadrado en el caso de no disponer de un `<slot>`. Si se activa `loading` el ícono se reemplazará por un spinner.

Puedes encontrar todos los iconos disponibles en [FontAwesome](https://fontawesome.com/search?q=)

<Canvas>
  <Story name="Icon" argTypes={renderArgs(KButton, {}, true)} parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        components: { KButton },
        template: `
          <KButton class="mr-2" color="primary" icon="user">Con texto</KButton>
          <KButton class="mr-2" color="primary" icon="user" />
          <KButton color="primary" icon="user" loading>Con texto</KButton>
        `,
      };
    }}
  </Story>
</Canvas>
