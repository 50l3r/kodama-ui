# KConfirm
Con el componente `k-confirm` puedes pedir una confirmación previa al usuario al realizar una acción importante.

<Story name="Default" parameters={renderParameters({showPanel: true})}>
  {Template.bind({})}
</Story>

<ArgsTable story="Default" />

## **Titulo**
El estilo neón aplica un efecto de iluminación de la alerta bastante bonico.

<Canvas>
  <Story name="Title" argTypes={renderArgs(KConfirm, {}, true)} parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        components: { KConfirm },
   
        template: `
          <k-confirm title="Vas a realizar una locura tremenda ¿Vamos con ello?">
            <k-button color="danger">Eliminar</k-button>
          </k-confirm>
        `,
      };
    }}
  </Story>
</Canvas>

## **Icono**
Modifica el icono por defecto del mensaje de confirmacón.

<Canvas>
  <Story name="Icon" argTypes={renderArgs(KConfirm, {}, true)} parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        components: { KConfirm },
   
        template: `

        `,
      };
    }}
  </Story>
</Canvas>

## **Botones**
Puedes personalizar el texto de los botones de confirmación con `okText` y `koText`.

<Canvas>
  <Story name="Buttons" argTypes={renderArgs(KConfirm, {}, true)} parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        components: { KConfirm },
   
        template: `
          <k-confirm ok-text="OK" ko-text="KO">
            <k-button color="danger">Eliminar</k-button>
          </k-confirm>
        `,
      };
    }}
  </Story>
</Canvas>
