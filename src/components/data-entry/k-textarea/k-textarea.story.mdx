Puedes establecer distintos tamaños que van desde `xs` hasta `lg`.

## **Errors**

<Canvas>
  <Story name="Errors" argTypes={renderArgs(KTextarea, {}, true)} parameters={renderParameters({showPanel: false})}>
    {() => {
      return {
        components: { KTextarea },
        setup() {
          return { errors : [{$message: 'Lorem ipsum dolor sit'}] };
        },
        template: `
          <k-card>
            <k-textarea :errors="errors" :width="400" :min-height="100">Label MD</k-textarea>

</Canvas>
